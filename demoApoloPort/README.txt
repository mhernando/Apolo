Utilizacion

La demo se utilizara para manejar elementos de Apolo desde un porgrama externo a este.

El programa pedira:

Introudir comando:
	Se introducira p, j o g dependiendo la eleccion.

Se dira el nombre del mundo en el que se encuentra el objeto a manipular. Se introducira
0 si se desea utilizar el primer objeto que se encuentre con el nombre dado
independientemente del mundo

Se seleccionara el nombre del objeto

A continuacion pedira los datos necesario para desplazarlo:

Posicion

	Pedira posicion: Introducir por consola las 3 coordenadas x y z 
			 separadas por espacios
	Pedira orientacion: Introducir por consola las 3 coordenadas r p yw
			    separadas por espacios

Mover joints

	Se introducira el numero de joints a mover
	Se introducira que joint se desea mover
	Se introducira nueva posicion del joint (una sola coordenada)

Obtener datos del sensor
	
	Obtendra los datos del sensor seleccionado
